# Telegram Gift App client

## Tech
- React;
- TypeScript;
- react-i18next;
- lottie-react;

## Launch

`npm install`

### dev
`npm run dev`

### build
`npm run build`

Before launch, you have to create `.env.development.local` and `.env.production.local` in the root. It must contain `PUBLIC_API_URL=https://api.url`
